================================================================================
  DÉPLOIEMENT
================================================================================

TEMPS REQUIS: 5-10 minutes

ÉTAPES RAPIDES:
1. Serveur: apt update && apt install nginx certbot
2. Fichiers: Copier dans /var/www/ljt-paysage/
3. Config: sudo cp nginx.conf /etc/nginx/sites-available/ljt-paysage
4. Symlink: sudo ln -s sites-available/ljt-paysage sites-enabled/
5. Test: sudo nginx -t && sudo systemctl reload nginx
6. SSL: sudo certbot certonly --nginx -d holua.duckdns.org
7. Vérify: curl -I https://holua.duckdns.org/

VOIR DEPLOYMENT.md pour guide détaillé.
