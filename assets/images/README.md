## ✅ Bonnes Pratiques

### Nommage des Fichiers
```
✅ BIEN : logo-ljt-principal.png
✅ BIEN : gallery-jardin-moderne-01.jpg  
✅ BIEN : team-laurent-dupont.jpg
❌ ÉVITER : IMG_1234.jpg
❌ ÉVITER : photo finale.PNG
```

### Optimisation
- **Taille max :** 2MB par image
- **Résolution :** Max 1920px de large
- **Qualité JPG :** 85-90%
- **PNG :** Uniquement si transparence nécessaire

### SEO Images
- Ajouter des noms descriptifs
- Utiliser des alt-text pertinents
- Compresser avant upload
