## 🚀 Quick Start

```bash
npm install
npm run build
npm run dev
```

## 📸 Gestion des Images

### Upload d'Images
- **Dossier principal :** `assets/images/`
- **Sous-dossiers organisés :**
  - `logo/` → Logos LJT Paysage
  - `gallery/` → Photos projets/réalisations  
  - `team/` → Photos équipe
  - `icons/` → Icônes interface

### Formats Recommandés
- **Photos :** JPG (optimisé web, max 1920px)
- **Logos/Icônes :** PNG ou SVG
- **Taille max :** 2MB par image

### Nommage
```
logo-ljt-main.png
gallery-projet-01.jpg
team-laurent-dupont.jpg
icon-phone.svg
```
